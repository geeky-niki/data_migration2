# 📦 Data Load

## 📖 Getting Started

1. **Clone the Repository:**
   ```bash
   git clone https://github.com/your-username/data_migration2.git

2. **Navigate through script directory:**
   ```bash
   cd data_migration2/python_scripts
3. **Run a Script:**
   ```bash
   python your_script.py
